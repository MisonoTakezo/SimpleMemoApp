### 動かし方

1 このリポジトリをクローン\

2 パッケージをインストール
```
$ npm install
```
3 Reactプロジェクトを実行
```
$ npm start
```
4 以下のURLにアクセス\
[http://localhost:3000](http://localhost:3000)
